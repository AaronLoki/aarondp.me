# Images Directory

This directory contains images used throughout the IB Grade Study website.

## Organization

Images should be organized by subject:

```
images/
├── physics/
├── chemistry/
├── biology/
├── math/
└── general/
```

## File Types

- PNG (preferred for diagrams and screenshots)
- JPG (for photos)
- SVG (for vector graphics)
- GIF (for animations if needed)

## Guidelines

- Use descriptive file names
- Optimize images for web (compress before uploading)
- Use appropriate formats (PNG for transparency, JPG for photos)
- Keep dimensions reasonable (max 2000px width)

See [UPLOAD_GUIDE.md](/UPLOAD_GUIDE.html) for detailed instructions.
