# Physics Files

Upload Physics past papers, notes, and other resources to this directory.

## Naming Convention

- Past papers: `paperX-[month]-[year].pdf` (e.g., `paper1-may-2023.pdf`)
- Notes: `topic-name-notes.pdf` (e.g., `mechanics-notes.pdf`)
- Questions: `topic-name-questions.pdf` (e.g., `electricity-questions.pdf`)

For detailed upload instructions, see [UPLOAD_GUIDE.md](/UPLOAD_GUIDE.html).
