# Files Directory

This directory contains downloadable files for the IB Grade Study website.

## Organization

Files should be organized by subject and level:

```
files/
├── physics/
├── physics-sl/
├── chemistry/
├── chemistry-sl/
├── biology/
├── math-aa/
└── ...
```

## File Types

- PDF documents (past papers, notes, study guides)
- Word documents (editable resources)
- Excel files (data, calculators)
- Other educational resources

## Guidelines

- Keep file names descriptive and lowercase
- Use hyphens instead of spaces
- Keep file sizes reasonable (under 25MB preferred)
- Ensure you have permission to share the content

See [UPLOAD_GUIDE.md](/UPLOAD_GUIDE.html) for detailed instructions.
